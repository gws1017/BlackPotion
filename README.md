>💡 블랙잭 게임의 룰을 이용해 포션을 제작하는 간단한 게임입니다.
플레이어는 포션을 제작하여 매일 늘어나는 목표 금액을 채우면서, 모은 재화로 더 높은 가격의 포션 레시피를 해금하고, 더 많은 금액을 벌면서 최대한 오래 버티는 것이 목표입니다.
유니티 사용 경험이 적어서 유니티를 사용해 간단한 게임을 만들어보기 위해 참여하게 되었습니다.
